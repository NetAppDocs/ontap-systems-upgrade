= Set the FC or UTA/UTA2 configuration on node3
:hardbreaks:
:nofooter:
:icons: font
:linkattrs:
:imagesdir: ../media/

[.lead]
include::../_include/ru_auto_set_fc_or_uta_uta2_confgig_node3_intro.adoc[]

If node3 doesn't have onboard FC ports, onboard UTA/UTA2 ports, or a UTA/UTA2 card, and you are upgrading a system with storage disks, you can skip to link:reassign-node1-disks-to-node3.html[Reassign node1 disks to node3].

include::../_include/ru_auto_config_fc_ports_node3.adoc[]

[start=8]
. Perform one of the following actions:
+
* If node3 has a UTA/UTA2 card or UTA/UTA2 onboard ports, go to <<Check and configure UTA/UTA2 ports on node3>>.
* If node3 doesn't have a UTA/UTA2 card or UTA/UTA2 onboard ports, skip <<Check and configure UTA/UTA2 ports on node3>> and go to link:reassign-node1-disks-to-node3.html[Reassign node1 disks to node3].

include::../_include/ru_auto_check_config_uta_uta2_ports_node3_intro_step_11.adoc[]

[start=10]
[[auto_check3_step10]] 
include::../_include/ru_auto_check_config_uta_uta2_ports_node3_steps_13.adoc[]

. Boot the node into boot menu by running `boot_ontap menu`. 

.What's next?

* If you are upgrading to an AFF A800 system, go to link:reassign-node1-disks-to-node3.html#auto_check3_step9[Reassign node1 disks to node3, Step 9].

* For all other system upgrades, go to link:reassign-node1-disks-to-node3.html[Reassign node1 disks to node3, Step 1].
