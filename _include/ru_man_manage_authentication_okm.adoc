You can use the onboard key manager to manage authentication keys. If you plan to have an onboard key manager (OKM) set up, you must record the passphrase and backup material before the beginning the upgrade.

.Steps

. Record the cluster-wide passphrase
. Back up the key-manager information:
+
`security key-manager onboard show-backup`
